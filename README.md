# 📚 Library App

A lightweight Library Management App built using only HTML, CSS, and Vanilla JavaScript. Users can view added books, add more books or remove them, and change read status.

## ✨ Features

📖 View Available Books.

➕ Add New Books.

✅ Change read status

❌ Remove books.
